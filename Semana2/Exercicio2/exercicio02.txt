Exercício 2:
a) Liste e descreva o que são as 4 etapas do processo de compilação.

•Preprocessing -> código fonte é incluído, macros ou definições são substituídos e processados, comentários são processados.

	•Compilation Proper -> código fonte é transformado em código assembly ( código que se assimila ao código de uma máquina/ binário).

	•Assembly  -> o assembler pega o código assembly e o converte, se tornando em um código de fato binário.

	•Linking -> as bibliotecas são adicionadas no código binário, e assim, cria-se um arquivo binário executável.
